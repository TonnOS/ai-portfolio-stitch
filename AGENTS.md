# AI Portfolio Stitch

Cyberpunk AI Developer Portfolio - Generated with Google Stitch, built with Vue 3 + Tailwind CSS.

## Tech Stack

- **Vue 3** + Vite
- **Tailwind CSS** 
- **Google Stitch** (Design Generation)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

GitHub Pages is configured with base path `/ai-portfolio-stitch/`.

## Initialize with Opencode

```bash
opencode init
```

This will set up the project for AI-assisted development with Opencode.
